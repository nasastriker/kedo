API лицензия

@@include('components/bem-blocks/_button.html',{ "color": "green", "text": "Подключить" })

Диадок API (Application Programming Interface) представляет собой интеграцию между системой электронного документооборота «Диадок» от компании Kontur и другой учетной системой. В том числе с нетиповой или имеющей индивидуальные доработки. Взаимодействие программных компонентов осуществляется с помощью набора определений и протоколов.

Преимущества API Diadoc

Интеграция через API — универсальна для любого бизнеса. Она будет незаменима если вам требуется создать уникальное решение под конкретные задачи компании. Или же в том случае, если нет типового решения ЭДО для той информационной системы, в которой работаете вы.

Среди основных преимуществ API Диадок также можно выделить умение интегрироваться с ПО, написанном на любом известном языке программирования, запускаемом как в операционной системе Microsoft Windows, так и в Linux.

Возможности API Диадок

Система ЭДО Диадок обладает широким спектром возможностей, которые упрощают работу с документами и делают взаимодействие организации с контрагентом более эффективным. Перечислим основные функции Диадока от компании Контур:

Создание документов, соответствующих утвержденному Федеральной налоговой службы формату xml;

Настройка цифровой сертифицированной подписи;

Передача одного или нескольких документов как в рамках своей информационной системы, так и в другие информационные системы, которыми пользуются контрагенты;

Отслеживание статусов документов, возможность их аннулирования, создание запросов на корректировку, создание черновика документа;

Удобное управление списком активных контрагентов с помощью фильтров;

Поиск контрагентов по реквизитам.

Учитывайте, что при разработке интеграционного решения разработку придется делать с нуля. Это необходимо для того, чтобы API Диадок полностью соответствовал вашим требованиям к электронному документообороту и вписывалась в интерфейс информационной системы, которой вы пользуетесь. Также при работе с API Диадоком необходимо соблюдать требования законодательства.

Что касается Контур.Диадок, то компания предлагает два вида интеграции ЭДО через API. Это HTTP-based API и AddInDiadocAPI. В первом случае объединение происходит с произвольными учетными записями. Во втором — с учетными записями на базе продуктов 1С.
